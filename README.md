# KairoCoin

KairoCoin is an open, decentralized blockchain protocol designed for secure
digital value transfer and long-term sustainability.

The project focuses on predictable monetary policy, validator accountability,
and gradual, responsible evolution of protocol features.

## Key Properties

- Fixed total supply: 1,000,000 KAIRO
- Transparent genesis allocation
- Proof-of-Stake inspired consensus
- Cryptographically verifiable state
- Open-source development

## Status

KairoCoin is under active development.
This repository currently contains the protocol whitepaper and foundational
documentation.

## Philosophy

KairoCoin prioritizes correctness, transparency, and long-term trust over
short-term speculation.
